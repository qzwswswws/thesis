\begin{cabstract}
面向辅助交互与便携式使用场景，脑机接口系统往往需要兼顾穿戴便利性、无线通信、持续运行与本地响应能力，因此难以直接沿用实验室中的高密度有线配置。由此带来的少导联采集、有限算力和在线交互约束，使得运动想象脑电解码任务需要从标准离线设置重新收束到更可部署的系统方案。针对这一问题，本文围绕双导端侧脑机接口原型，系统开展了硬件实现、低通道解码方法设计与在线联动验证研究，探索一条由标准高密度离线解码走向便携式少导联系统实现的技术路径。

本文首先设计并实现了以 KS1092 双通道专用前端与 nRF52832 主控为核心的脑电采集硬件系统，完成了脑电采样、参数配置、BLE 无线传输和上位机显示链路的搭建，并通过 Alpha 节律验证与标签链路响应实验确认了系统具备基本的神经生理信号采集与事件标记能力。在算法方面，本文提出了面向低通道端侧场景的改进型自注意力脑电解码方法 EdgeMIFormer，并在统一协议下完成全通道、少通道和双导条件下的对照验证，据此确定了更适合双导系统的任务收束路径与模型配置。此外，模型经剪枝与 8-bit 量化后，参数规模压缩至原始尺寸的 47.9\%，并在 RK NPU 环境下实现约 31\,ms 的单次推理时间，完成了端侧部署验证。

在系统集成层面，本文进一步构建了“预训练初始化 + 轮次级模型更新 + 跨轮次判别反馈”的在线联动机制。按时间顺序组织的伪在线评估结果表明，有限参数重拟合比单纯在线对齐更具潜力；真实双导左右手在线试跑则进一步说明，所提出的系统已经具备从采集、记录、训练到跨轮次推理反馈的基本闭环能力。本文已形成从标准高密度离线解码到双导端侧在线原型的初步落地路径。研究结果可为面向辅助交互与便携式使用场景的低成本脑机接口系统，在任务收束、低通道模型设计、端侧部署实现和在线联动机制构建方面提供参考。

\end{cabstract}

\begin{eabstract}
For assistive interaction and portable usage scenarios, brain-computer interface systems are expected to jointly satisfy wearing convenience, wireless communication, sustained operation, and local response. Under such requirements, it becomes difficult to directly retain the laboratory-style high-density wired configuration. The resulting few-channel acquisition, limited computing budget, and online interaction constraints require motor imagery EEG decoding to be reformulated from the standard offline setting into a more deployable system solution. To address this problem, this thesis focuses on a dual-channel edge-oriented BCI prototype and studies hardware implementation, low-channel decoding methodology, and online integration, aiming to derive a practical path from standard high-density offline decoding to a portable few-channel system.

First, a dual-channel EEG acquisition system was designed and implemented with the KS1092 front-end and the nRF52832 controller as the core components. The system supports signal acquisition, parameter configuration, BLE transmission, and host-side visualization. Alpha-rhythm verification and marker-chain response experiments confirmed that the platform can acquire physiologically meaningful EEG signals and support event tagging for online paradigms. Second, an improved self-attention EEG decoding model, EdgeMIFormer, was developed for low-channel edge scenarios. Under a unified protocol, comparative experiments were conducted in full-channel, reduced-channel, and dual-channel settings, based on which a more suitable task-constraining path and model configuration for the dual-channel system were determined. In addition, after pruning and 8-bit quantization, the model size was reduced to 47.9\% of the original, and the inference latency was about 31\,ms on an RK NPU, verifying edge-side deployment.

Finally, an online integration mechanism based on pretrained initialization, round-wise model updating, and cross-round prediction feedback was established. Chronological pseudo-online experiments indicate that limited parameter refitting is more promising than using online alignment alone, while real dual-channel left-right online runs further show that the system can already support a basic loop of acquisition, logging, training, and cross-round inference feedback. The thesis has therefore formed a preliminary path from standard high-density offline decoding to a dual-channel edge-oriented online prototype. The results provide useful evidence for task constraining, low-channel model design, edge deployment, and online integration in low-cost EEG-based BCI systems for assistive and portable scenarios.

\end{eabstract}
