## 4.3 四分类通道退化实验

### 4.3.1 实验目标

在完全不改变数据集、划分方式、标准化规则和评价指标的前提下，仅改变输入通道数，观察模型从全通道降到低通道后的性能退化轨迹。

### 4.3.2 对应脚本与结果

- 脚本：`baselines/conformer_degradation.py`
- 运行器：`baselines/run_degradation.py`
- 汇总文件：`results_summaries/degradation_4class.csv`
- 图表：`visualization/degradation_curve_4class.png`

本实验考察了四种通道设置：

- `full`：`22` 导
- `central8`：`8` 导
- `c3czc4`：`3` 导
- `c3c4`：`2` 导

### 4.3.3 结果汇总

| 通道配置 | 导数 | Avg Best Acc | Avg Aver Acc |
| --- | --- | --- | --- |
| full | `22` | `0.734189` | `0.600478` |
| central8 | `8` | `0.659722` | `0.551700` |
| c3czc4 | `3` | `0.591056` | `0.517033` |
| c3c4 | `2` | `0.497289` | `0.444067` |

相对全通道 `Best Acc` 的下降幅度约为：

| 对比 | 下降值 |
| --- | --- |
| `22 -> 8` | `-7.45` 个百分点 |
| `8 -> 3` | `-6.87` 个百分点 |
| `3 -> 2` | `-9.38` 个百分点 |
| `22 -> 2` | `-23.69` 个百分点 |

### 4.3.4 主要结论

- 在同一数据集、同一协议下，通道数减少与性能下降之间存在明确的单调关系。
- `8` 导仍保留了较强的可用性。
- `3` 导已经出现明显退化，但仍保有研究价值。
- `2` 导 `4分类` 明显进入高风险区，已经不适合作为工程落地主线。

### 4.3.5 论文中的角色

这组实验非常适合作为论文中的“证据链核心图表”，它能回答：

- 低通道问题不是拍脑袋设想，而是可量化、可重复观测的真实退化现象。
- 后续为什么需要任务重定义、结构优化或蒸馏补偿，不是因为“想换题”，而是因为 `2导 / 4分类` 本身确实困难。

## 4.4 双导二分类基线实验

### 4.4.1 实验目标

在双导 `C3/C4` 条件下，将任务从 `4分类` 重定义为左右手 `2分类`，验证这种任务重定义能否在不引入任何额外通道信息的前提下显著恢复识别性能。

### 4.4.2 对应脚本与结果

- 脚本：`baselines/conformer_degradation.py`
- 汇总文件：`results_summaries/degradation_2class.csv`
- 图表：`visualization/c3c4_4class_vs_2class.png`

### 4.4.3 结果

`C3/C4` 双导 `2分类` 的 `9` 被试结果为：

| 指标 | 数值 |
| --- | --- |
| Avg Best Acc | `0.707562` |
| Avg Aver Acc | `0.648781` |

与 `2导 / 4分类` 相比，提升约为：

| 指标 | 提升值 |
| --- | --- |
| Best Acc | `+21.03` 个百分点 |
| Aver Acc | `+20.47` 个百分点 |

### 4.4.4 主要结论

- 在双导输入下，任务重定义比盲目继续坚持 `4分类` 更有效。
- 该提升并非来自教师模型、额外通道或数据泄露，而是来自任务本身与感兴趣脑区生理机制的一致性增强。
- 因此，`2导 / 4分类` 更适合被定位为“低通道退化证据”，而 `2导 / 2分类` 更适合被定位为“工程落地入口”。

### 4.4.5 论文中的角色

这部分可以作为工程方案合理性的关键支撑：

- 当硬件被压缩到双导级别时，继续保持完整四分类目标会导致显著性能损失。
- 若最终系统目标更偏向左右手控制，则将任务收束为双导二分类具有生理依据和实验依据的双重合理性。

## 4.5 lowchannel_v1：大改版低通道原生模型探索

### 4.5.1 实验目标

尝试通过更激进的结构重写，构建一个更“低通道原生”的模型，包括：

- 多尺度时间卷积
- 晚期融合
- `C3/C4` 派生特征分支
- attention pooling 头

### 4.5.2 对应脚本与结果

- 设计说明：`00_AI_Management/Output_Drafts/Low_Channel_Model_Modification_Rationale.md`
- 脚本：`baselines/conformer_lowchannel_v1.py`
- pilot 运行器：`baselines/run_lowchannel_v1_pilot.py`
- 汇总文件：`results_summaries/lowchannel_v1_pilot_20260329_012906.csv`

pilot 被试固定为：

- `S1`
- `S3`
- `S5`
- `S8`
- `S9`

实验设置包括：

- `c3c4 / 2-class`
- `c3czc4 / 4-class`
- `c3c4 / 4-class`

### 4.5.3 结果

| 设置 | Pilot Avg Best | Pilot Avg Aver | 对应 baseline pilot |
| --- | --- | --- | --- |
| `c3c4 / 2-class` | `0.716667` | `0.649128` | `0.750000 / 0.697506` |
| `c3czc4 / 4-class` | `0.477778` | `0.426781` | `0.633340 / 0.562900` |
| `c3c4 / 4-class` | `0.518750` | `0.464414` | `0.534700 / 0.484140` |

其中主评价任务 `c3c4 / 2-class` 对 baseline 的被试级比较为：

- `Best Acc` 改善：`0 / 5`
- `Aver Acc` 改善：`0 / 5`

### 4.5.4 结论

- `lowchannel_v1` 未能在任何核心设置上形成稳定增益。
- 尤其在 `3导 / 4分类` 条件下，性能下降明显，说明其大规模结构重写反而破坏了原本尚可用的退化结果。
- 这一结果直接推动后续策略从“大改版”切换到“分步、单变量、小批次”的结构优化路线。

### 4.5.5 论文中的建议定位

这条线不适合作为“提出的新模型显著优于 baseline”的主结论。  
更适合写成：

- 一次探索性结构改造尝试。
- 结果显示，在低通道 EEG 解码中，同时叠加多种结构变化会显著增加变量耦合，反而不利于稳定提升。
- 因而后续改进路线改为更严格的分步消融。

## 4.8 最小知识蒸馏实验

### 4.8.1 实验目标

验证在 `2导 / 4分类` 条件下，是否可以通过训练阶段引入 `22导` teacher，从而把一部分高密度空间判别知识迁移给低通道 student。

该实验的定位不是“证明纯双导模型本身已足够强”，而是：

- 当 `2导 / 4分类` 出现明显性能断崖时，
- 训练期使用额外高密度信息，
- 能否作为一种补偿性技术路线恢复部分性能。

### 4.8.2 对应脚本与结果

- 脚本：`baselines/conformer_kd_student.py`
- 运行器：`baselines/run_kd_pilot.py`
- 汇总文件：`results_summaries/kd_pilot_20260329_112640.csv`

实验设置：

- teacher：`22导 / 4分类` baseline
- student：原始 `c3c4 / 4分类` Conformer
- 损失：`CE + KL`
- 温度：`2.0`
- `alpha = 0.5`
- pilot 被试：`S1, S3, S5, S8, S9`

### 4.8.3 结果

| 指标 | KD pilot | 对应 `2导 / 4分类` baseline pilot |
| --- | --- | --- |
| Avg Best Acc | `0.547222` | `0.534700` |
| Avg Aver Acc | `0.493058` | `0.484140` |

teacher 在这 `5` 个被试上的平均准确率为：

- `Avg Teacher Acc = 0.780555`

被试级对比结果：

- `Best Acc` 数值提升：`5 / 5`
- `Aver Acc` 数值提升：`5 / 5`

但其中 `S8` 的 `Best Acc` 提升极小，几乎可视为持平，因此更稳妥的表述是：

- KD 在 `5` 个 pilot 被试上表现出一致但幅度有限的正向增益。

### 4.8.4 结论

- 与前两条结构优化路线不同，KD 至少在当前 pilot 上是“有效的”。
- 但其提升幅度总体较温和，还不足以把 `2导 / 4分类` 直接拉回到与 `8导` 或 `22导` 接近的量级。
- 因此，KD 适合作为补充实验或补偿实验，而不适合作为主线叙事中“纯双导方案本身很强”的证据。

### 4.8.5 论文中的建议定位

建议将 KD 写成：

- 训练期特权信息下的补偿实验
- 或“离线高密度标定 -> 在线低通道部署”的工程范式补充

而不建议写成：

- 纯双导条件下不依赖额外信息即可达到相同效果

换句话说，KD 的论文价值是存在的，但它更适合“补充性创新点”，不宜吞掉主线。

## 5. 全部实验的综合结论

截至目前，`MI_Algorithm_Workbench` 内已经形成了一个比较清楚的研究链条：

### 5.1 已经站稳的结论

1. 在 `BCI IV 2a` 上，`22导 / 4分类` baseline 已被锁定，整体平均 `Best Acc` 约为 `73.42%`。
2. 在相同协议下，通道数从 `22 -> 8 -> 3 -> 2` 减少时，`4分类` 性能呈明显单调下降。
3. `2导 / 4分类` 虽仍高于随机水平，但已明显不适合作为最终工程落地主线。
4. 将双导任务重定义为 `左右手二分类` 后，性能显著回升到 `70.76%` 左右，说明双导场景下任务重定义比硬撑四分类更合理。

### 5.2 已经得到负结论的方向

1. 大改版低通道原生模型 `lowchannel_v1` 没有带来稳定收益。
2. 分步轻改的 `Batch 1` 和 `Batch 2` 也没有超过原始双导二分类 baseline。
3. 因而，“继续通过小规模结构补丁提升双导二分类”的边际收益已经偏低。

### 5.3 已经显示潜力、但应保持克制表述的方向

1. `22导 teacher -> 2导 student` 的最小 KD pilot 在 `2导 / 4分类` 上取得了一致但幅度有限的正向提升。
2. 这说明训练期利用额外高密度信息，的确有机会补偿双导四分类性能损失。
3. 但它不应被表述为“纯双导系统本身无需额外信息就能达到更高性能”。

## 6. 后续论文写作时的建议叙事

## 6.1 最稳的主线叙事

当前最适合作为论文主线的叙事顺序是：

1. 先给出 `22导 / 4分类` baseline。
2. 再给出 `22 -> 8 -> 3 -> 2` 导四分类退化曲线。
3. 说明 `2导 / 4分类` 在同协议下明显失稳。
4. 接着引出 `C3/C4 / 2分类` 的工程落地逻辑与显著性能恢复。
5. 最后补充说明：若仍希望在双导条件下维持四分类，可尝试 KD 作为训练期补偿手段。

这一顺序的优点是：

- 逻辑清楚
- 变量控制干净
- 主线结论不依赖蒸馏
- KD 可以自然作为增强段落或补充实验

## 6.2 不建议的写法

不建议在论文中把下面这些内容写成主线结果：

- `lowchannel_v1` 被描述为显著优于 baseline
- `Batch 1/2` 被描述为成功优化低通道
- 仅凭历史 `early_stop_*` 表格直接推导当前主线结论
- 将 KD 写成“纯双导模型性能本来就可以这样高”

## 7. 可直接转写为论文的结论草稿

以下文字可作为后续论文成稿时的中间素材。

### 7.1 关于通道退化的结论草稿

在 `BCI Competition IV 2a` 数据集上，本文首先固定 `A0xT` 训练、`A0xE` 测试、单被试建模及训练集统计量标准化测试集的统一协议，并以 `22` 导 Conformer 模型作为全通道基线。在此基础上，仅改变输入通道数，构建了 `22导`、`8导`、`3导` 与 `2导` 四种四分类设置。实验结果表明，随着通道数持续减少，平均识别性能呈现明确的单调下降趋势：`22导`、`8导`、`3导` 与 `2导` 的平均最佳准确率分别为 `73.42%`、`65.97%`、`59.11%` 与 `49.73%`。这一结果说明，低通道约束会显著削弱多类别运动想象识别性能，尤其在双导条件下，四分类任务已进入高风险区。

### 7.2 关于双导二分类落地性的结论草稿

考虑到双导 `C3/C4` 条件下四分类结果显著下降，本文进一步在相同数据集、相同训练测试协议以及相同双导输入条件下，将任务重定义为左右手二分类。结果显示，`C3/C4` 双导二分类的平均最佳准确率恢复至 `70.76%`，相较双导四分类提升约 `21` 个百分点。该现象表明，在极低通道硬件约束下，任务重定义是一条比单纯维持完整四分类更合理的工程路径，也与中央感觉运动区对左右手侧化活动更敏感的生理事实相一致。

### 7.3 关于低通道结构优化失败经验的结论草稿

为进一步提升低通道性能，本文尝试了低通道原生大改版结构以及两轮轻量结构消融，包括 attention pooling 头替换和 `C3-C4` 差分分支引入。然而，相关 pilot 结果均未稳定超过原始双导二分类 baseline。这说明，在当前实验协议下，低通道性能瓶颈并非仅由分类头形式或浅层输入重组决定，过于激进或过于局部的结构改动均难以带来可靠增益。

### 7.4 关于知识蒸馏补偿实验的结论草稿

在保持双导 student 推理形式不变的前提下，本文进一步考察了 `22导 teacher -> 2导 student` 的最小知识蒸馏方案。以 `C3/C4` 双导四分类为对象，pilot 结果显示蒸馏后 student 在 `5` 个代表性被试上取得了一致但幅度有限的性能提升，平均最佳准确率由 `53.47%` 提升至 `54.72%`。这表明，当低通道四分类性能出现明显断崖时，训练期引入高密度教师模型能够对 student 起到一定的知识补偿作用，但该方法更适合作为补充实验或离线标定式工程范式，而不宜替代纯双导主线方案。
