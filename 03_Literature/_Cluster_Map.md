# Literature Cluster Map

This directory stores archived literature assets grouped by knowledge cluster.

## Cluster Layout

- `K1_MI_Physiology`
  - MI physiology papers, PDF full texts, HTML full texts, and local article asset folders.
  - Current examples: `Pfu06.pdf`, `Pfu97.pdf`, `Mu and beta rhythm...pdf`, `full-paper_873.pdf`, `Bla22.pdf`, and the saved HTML/article folders for `Zap20b`, `Xu19`, and `Men18`.
- `K2_Transformer_for_EEG`
  - Transformer-in-EEG evolution papers and source materials.
  - Current examples: `ACNN-Transformer...pdf`, `EEGConformer...pdf`, `Hierarchical_Transformer...pdf`.
- `K3_Efficient_Local_Attention`
  - Efficient/local attention literature and parsed text outputs.
  - Current examples: `Bus24.txt`, `Hua23b.txt`, `Keu24.txt`, `Son22.txt`, `Wan23.txt`.
- `K4A_Edge_Compression_Quantization`
  - Edge-model compression, quantization, and lightweight architecture papers.
  - Current examples: `MI-BMInet_...pdf`, `Reducing_False_Alarms...EEGformer...MCUs.pdf`.
- `K4B_Edge_Deployment_Platforms`
  - RK3568, embedded NPU, MCU, and on-device deployment papers.

## Usage Rule

- Put paper assets here first: PDF, HTML full text, saved article folders, parsed text, and other paper-centric materials.
- Keep active research reports and prompt-routing files in `00_AI_Management/Input_Buffer`.
- When a paper becomes part of the thesis working set, place it directly into the matching cluster here.
