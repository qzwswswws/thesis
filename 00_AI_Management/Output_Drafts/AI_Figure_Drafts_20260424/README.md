## 不建议直接交给 AI 定稿的图

| 图类型 | 原因 |
| --- | --- |
| 在线误差分析图 | 需要严格对应真实运行日志与统计结果 |
| 部署折中关系图 | 本质是数据图，应该基于表格和脚本重绘 |
| 实物样机与佩戴/电极布置图 | 更适合真实拍摄或基于实拍图后期标注 |
