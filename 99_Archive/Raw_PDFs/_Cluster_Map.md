# Raw PDF Cluster Map

This directory is now a legacy placeholder. Active thesis PDFs have been moved into `03_Literature` under the same knowledge-cluster scheme.

## Cluster Layout

- `K1_MI_Physiology`
  - Reserved for raw MI-physiology PDFs.
- `K2_Transformer_for_EEG`
  - Raw PDFs for Transformer-in-EEG evolution papers.
  - Current examples: `ACNN-Transformer...pdf`, `EEGConformer...pdf`, `Hierarchical_Transformer...pdf`.
- `K3_Efficient_Local_Attention`
  - Raw PDFs for efficient/local attention and lightweight attention variants.
  - Current examples: sliding-window attention, Swin-channel-attention, compact convolutional transformer, and the efficient-transformers report PDF.
- `K4A_Edge_Compression_Quantization`
  - Raw PDFs for compact deployment, MCU-oriented models, quantization, and compression.
  - Current example: `Reducing_False_Alarms...EEGformer...MCUs.pdf`.
- `K4B_Edge_Deployment_Platforms`
  - Reserved for raw PDFs focused on hardware platforms, NPU deployment, or device-level benchmarking.

## Storage Rule

- New thesis-related PDFs should be stored directly in `03_Literature` by knowledge cluster.
- Keep this directory only for exceptional legacy fallback storage when a file should not enter the active literature set yet.
